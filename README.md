# Valentine Web Page

Мини-сайт из нескольких экранов с интерактивом: вступление, причины любви, игра «найди пары», коллаж воспоминаний и финальное поздравление с 8 Марта.

## Как запустить

1. Открой `index.html` в браузере.
2. Для обновления после изменений сделай жесткую перезагрузку (`Ctrl+F5`).

## Структура проекта

- `index.html` — вся разметка экранов и основных блоков.
- `style.css` — стили, адаптив, анимации.
- `script.js` — логика экранов, игры, таймера, модалки фото и фоновых сердечек.
- `images/` — фотографии для коллажа воспоминаний.

## Как работает сайт

### 1) Экран приветствия

- Заголовок и кнопка `Начать игру`.
- Блок `Сколько мы вместе` считает дни от даты из:
  - `data-start-date="2024-10-25T20:00:00"`.
- При первой загрузке показывается стартовая анимация сердечка.

### 2) Экран «Почему я тебя люблю»

- Список карточек причин.
- Слева сердечко-чекбокс:
  - `🤍` — не выбрано,
  - `💗` — выбрано.
- При выборе раскрывается текст причины.

### 3) Игра «Найди пары»

- Карточки перемешиваются случайно.
- Считаются ходы и найденные пары.
- Кнопка перехода к воспоминаниям активируется только после нахождения всех пар.
- На очень маленьких экранах (`<= 420px`) игра автоматически упрощается до 4 пар (вместо 6).

### 4) Коллаж воспоминаний

- Фото берутся из `images/*.jpg` по путям в `index.html`.
- Карточки стилизованы под Polaroid.
- По клику фото открывается в полноэкранной модалке.
- Закрытие модалки: кнопка `✕`, клик по фону или `Esc`.

### 5) Финальный экран

- Экран поздравления с 8 Марта.
- Переход на него кнопкой `К поздравлению →` под коллажем.

## Фоновые и UI-анимации

- Падающие сердечки фоном на всех экранах.
- Плавное появление панелей.
- Анимация совпавших карточек в игре.
- Пульсация сердечка на финальном экране.
- Есть поддержка `prefers-reduced-motion`.

## Оптимизация под устройства

Сделано для мобильных, планшетов и десктопа:

- Адаптивные брейкпоинты для типографики и сеток.
- Безопасные отступы (`safe-area`) для устройств с вырезами.
- Touch-оптимизация (`touch-action: manipulation`).
- Lazy loading изображений (`loading="lazy"`, `decoding="async"`).
- Пауза/ограничение фоновой анимации сердечек для снижения нагрузки.
- Блокировка прокрутки фона при открытой модалке фото.

## Как менять контент

### Изменить дату «сколько мы вместе»

В `index.html` найди:

```html
<div id="together-timer" data-start-date="2024-10-25T20:00:00">
```

И поставь свою дату в формате:

`YYYY-MM-DDTHH:mm:ss`

Пример: `2024-10-25T20:00:00`.

### Изменить фото в коллаже

1. Положи свои фото в папку `images/`.
2. В `index.html` обнови `src` у нужных `<img>`:

```html
<img src="images/1.jpg" alt="Воспоминание 1" loading="lazy" decoding="async">
```
